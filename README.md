# Uday Kajana CodeForge

This repository contains various coding projects and implementations across different technologies and domains.

## Branches Overview


### apachebeam
- **Purpose**: Dedicated to Apache Beam data processing framework
- **Contents**:
  - Java SDK examples with full pipeline implementations
  - Compiled Java classes
  - Python SDK workbench notebook
  - GCS to GCS data transfer example

### datastructures_and_algorithms
- **Purpose**: Comprehensive data structures and algorithms library
- **Contents**:
  - Linked list implementations (single, double, circular)
  - Tree structures (BST, AVL, Trie)
  - Graph algorithms
  - Hashing techniques
  - Sorting algorithms
  - Queue and stack implementations

### json_editor
- **Purpose**: JSON editing application
- **Contents**:
  - Java Swing-based JSON editor
  - Custom UI components (rounded buttons, text fields)
  - Property panels and indentation management
  - IntelliJ IDEA project configuration

### leet_code_solutions
- **Purpose**: Solutions to LeetCode coding problems
- **Contents**:
  - Java implementations of algorithmic problems
  - Dispatcher, Solution, and Toolkit classes

### master
- **Purpose**: Legacy branch (deprecated)
- **Contents**: Identical to main branch

### pyspark
- **Purpose**: PySpark and Dataproc examples
- **Contents**:
  - Jupyter notebook for Dataproc Spark workflows
  - Sample world population dataset (CSV)
  - Documentation and examples

### main (Default Branch)
- **Purpose**: Central branch containing a mix of all projects
- **Contents**:
  - Apache Beam Java SDK examples (combiners, pipelines, etc.)
  - Data Structures and Algorithms implementations (linked lists, trees, hashing, sorting)
  - LeetCode problem solutions in Java
  - Python scripts for data pipelines and Google Cloud Spanner
  - Miscellaneous files and documentation
